Wireframes & Prototyping

This project focused on designing user flows and wireframes to improve the overall usability of a digital product.

Problem

The existing interface lacked a clear navigation structure, making it difficult for users to complete key tasks efficiently.

Design Process

Created low-fidelity wireframes to explore layout solutions and interaction patterns. These were iterated into higher fidelity prototypes using Figma.

Prototype

Interactive prototypes were developed to simulate user flows and test navigation clarity.

Outcome

The final wireframes improved task flow and reduced unnecessary steps in the user journey.
